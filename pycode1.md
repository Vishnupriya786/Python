# Solve Me First
```
 Function prototype:

 int solveMeFirst(int a, int b);

 where,

 a is the first integer input.
 b is the second integer input
 Return values

 sum of the above two integers
```
```
Sample Input

a = 2
b = 3
```
```
Sample Output

5
```
```

def solveMeFirst(a,b):
	return a+b

num1 = int(input())
num2 = int(input())
result = solveMeFirst(num1,num2)
print(result)
```
